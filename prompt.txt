Buatkan saya homepage website ecommerce anime merchandise bernama Merchansuki menggunakan HTML + TailwindCSS CDN dalam satu file lengkap siap pakai. Desain harus modern, clean, premium anime store style, mobile-first dan fully responsive. Gunakan font Poppins dari Google Fonts dan konfigurasi warna dengan primary amber (#fbbf24), secondary blue (#3b82f6), accent red (#ef4444), maroon (#800000), background utama putih, serta section promo menggunakan gradient maroon gelap (#4c0519 → #800000 → #4c0519). Gunakan banyak rounded-xl dan rounded-2xl, soft shadow, smooth transition, hover animation halus, dan glass effect ringan pada product card.

Struktur homepage harus dimulai dengan maintenance notification bar full width di paling atas menggunakan gradient kuning ke orange ke merah dengan teks putih bold kecil dan shadow ringan. Di bawahnya terdapat navbar sticky dengan background biru, logo bulat di kiri dengan nama Merchansuki, menu desktop Home, Catalog Produk, dan About, serta di sisi kanan terdapat icon search, icon cart dengan badge merah kecil, dan tombol login atau user dropdown. Navbar harus responsive dengan hamburger menu di mobile dan animasi dropdown fade-down.

Setelah navbar tampilkan hero carousel banner full width dengan auto slide setiap 5 detik, tinggi responsive dari 200px di mobile hingga 600px di desktop, gambar menggunakan object-contain agar tidak terpotong, memiliki tombol prev dan next berbentuk bulat putih dengan shadow, serta dot indicator di bagian bawah tengah dengan animasi transisi transform yang smooth.

Di bawah carousel tampilkan section promo import dengan layout dua kolom (stack jadi satu kolom di mobile), background gradient maroon gelap, sisi kiri berisi gambar karakter besar dengan drop shadow dan efek hover scale, sisi kanan berisi badge kecil transparan, heading besar italic uppercase dengan highlight warna amber pada sebagian kata, paragraf deskripsi warna abu terang, dan tombol rounded full berwarna amber dengan efek glow shadow.

Selanjutnya buat section Recommended Collection dengan background abu terang, container max-width 7xl, heading besar uppercase dengan subheading italic dan garis aksen orange di bawah judul. Gunakan grid produk 2 kolom di mobile dan 4 kolom di desktop. Product card harus rounded-2xl dengan glass effect ringan, hover translateY naik sedikit, shadow saat hover, image slider otomatis jika memiliki beberapa varian (ganti gambar setiap 3 detik), badge kecil “Best Choice” warna merah, harga bold warna orange, dan tombol hitam yang berubah menjadi orange saat hover.

Terakhir buat footer dengan 4 kolom yaitu brand dan deskripsi dengan social media icon bulat, quick links, customer service links, dan newsletter form dengan input dan tombol subscribe. Gunakan background putih dengan border top, hover link berubah warna, dan bagian copyright di bawah dengan icon heart animasi pulse.

Tambahkan juga script vanilla JavaScript untuk toggle mobile menu, auto carousel 5 detik, manual prev/next carousel, update indicator aktif, dan slider varian produk 3 detik. Semua harus dalam satu file HTML lengkap termasuk Tailwind CDN, Google Fonts, dan FontAwesome CDN tanpa backend dan gunakan dummy data produk minimal 4 item.
